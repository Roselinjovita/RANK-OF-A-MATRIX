### Step 3:

### Step 4: 
End the program

## Program:
```python
#Program to find the rank of a matrix.
#Developed by: S.Roselin mary jovita
#RegisterNumber:212222230122
import numpy as np
A=np.array([[3,2,5],[1,1,2],[3,3,6]])
rank=np.linalg.matrix_rank(A)
print(rank)
```
